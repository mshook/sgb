## What are components in graphs

Three types relevant to SGB:

- **Connected components** (undirected) — maximal sets of mutually reachable vertices. Used by `word_components`.
- **Biconnected components** (undirected) — no articulation point; removing any single vertex leaves it connected. Used by `book_components` on literary character graphs.
- **Strongly connected components** (directed) — every vertex can reach every other in both directions. Used by `roget_components` on Roget's Thesaurus.

All three algorithms run in O(V+E). Biconnected and strongly connected both use DFS with Tarjan's low-link technique.

## Does Tufte talk about graphs

Tufte (*Visual Display of Quantitative Information*, etc.) covers statistical graphs (charts, plots) extensively. He barely touches graph theory / network graphs — occasionally shows network-like diagrams but as visual design problems, not mathematical structures. The two traditions (combinatorics/algorithms vs. visual communication) largely talk past each other.

Closest bridge: Bertin's *Sémiologie Graphique* (1967), which treats both statistical and relational/network graphics in a unified framework.

## cfmt — C indentation formatter (TDD)

Standard modern idiomatic C indentation: **4 spaces**.

Built a CLI C formatter (`cfmt.c`) following TDD:

5. Ran tests — 9/9 passed

### Algorithm

Line-by-line re-indentation with a lexer state machine (`NORMAL | IN_STRING | IN_CHAR | IN_LINE_COMMENT | IN_BLOCK_COMMENT`) that persists across lines to handle multi-line block comments.

Per-line logic:
```
pre_close = count of } before first { on line
open      = total { on line  (outside strings/comments)
close     = total } on line

indent -= pre_close          (clamped ≥ 0)
print (indent * 4 spaces) + stripped line
indent += open - (close - pre_close)   (clamped ≥ 0)
```

Correctly handles `} else {` (indent dips then rises), inline `{ }` (net zero), nested braces, braces in strings/comments (ignored).

### Usage

```sh
./cfmt file.c           # format a file → stdout
./cfmt < file.c         # from stdin
./cfmt a.c b.c          # multiple files concatenated
```

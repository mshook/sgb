# Sonnet

In Knuth's graphs, the nodes and edges speak,
Of words and miles and football seasons past,
Of Roget's web where synonym chains cascade,
And Mona Lisa's pixels hold steadfast.

The random flip, a seed from which all grows,
The spanning tree that links each city's mile,
Through biconnected books where plotline flows,
And Ramanujan's expanders rank and file.

In literate prose the C code finds its form,
Each `.w` file a chapter, tangle-spun,
The library compiled through make and norm,
A graph for every problem under sun.

  So Knuth composed, in C and CWEB's art,
  The Stanford GraphBase — algorithms heart.
